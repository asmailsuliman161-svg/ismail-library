# مكتبة إسماعيل سليمان حسن

مرحبًا بكم في المكتبة الرقمية التعليمية لإسماعيل سليمان حسن.  
تشمل المكتبة كتب المنهج، امتحانات، أوراق عمل وملخصات لجميع المراحل التعليمية:  
- المرحلة الابتدائية  
- المرحلة المتوسطة  
- المرحلة الثانوية  

## استخدام المكتبة
1. افتح المجلد الخاص بالمرحلة.  
2. اختر نوع الملف (كتاب / امتحان / ملخص).  
3. اضغط على الرابط لعرض الكتاب أو تحميله.

## مؤلف المكتبة
- الأستاذ إسماعيل سليمان حسن
- الخبرة: 17 سنة في التعليم وتخصص نظم المعلومات الإداري
